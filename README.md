# Alblog
A Light Blog

一个前后端分离的轻博客内容管理系统

## 实现技术

后端 API 基于 nodejs + mongodb + redis

前端页面基于 vue + vue-router + vuex + axios 

## 运行环境

NodeJS v6.0.0 + 

Mongodb v3.2.5 + 

Redis v3.2.100 +

## 已完成功能：

1、用户注册

2、用户登录

3、文章发布

4、文章查询

5、标签查询

6、文章管理

7、文章点赞

8、评论点赞

9、文章评论

10、评论回复

11、分类添加

12、分类管理

等等功能。
